 ### Election-Audit Summary
One way the script can be used is determine the popular candidate among the candidates. The script is used to ask a voter to make selection of a candidate from the candidate options in the election. The input of the voter is then interpreted by the script representing a candidate selected if on the list else it's added, the number of votes the candidate receives, and the candidate name. The selection the voter makes is then used to automatically compute the inforamtion like the voters count for each county, total votes cast, candidate with the most vote count, and so on.

Another way the script can be used is to analyze the demographics that vote the most in an election. Thus comparing between those that are voting to those that are not.
